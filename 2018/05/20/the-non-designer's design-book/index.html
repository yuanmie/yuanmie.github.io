写个大家看的设计书·浅读

发表于 2018-05-20 | 分类于 notes

设计的四个基本法则：

亲密性

彼此相关的项应当靠近，归为一类。如果多个项相互之间存在很近的亲密性，它们会称为一个视觉单元，而不是多个孤立的元素。这有助于组织信息，减少混乱，为读者提供清晰的结构。

对齐

任何东西都不能在页面上随意安放。每个元素都应当与页面上的另一个元素有某周视觉联系。这样能建立一种清晰、精巧而且清爽的外观。

重复

让设计中的视觉要素在整个作品中重复出现。可以重复颜色、形状、材质、空间关系、线宽、字体、大小和图片，等等。这样一来，既能增加条理性，还可以加强统一性。

对比

要避免页面上的元素太过相似。如果元素（字体、颜色、大小、线宽、形状、空间等）不相同，那么就干脆让他们截然不同。要让页面引人注目，对比通常是最重要的一个因素，正是它能使读者首先看这个页面。

颜色

色轮就是颜色组成的轮子😀。色轮的基础是黄、红和蓝3种颜色。

色轮相对的颜色称为互补颜色。

最佳实践：

双色：

  • 互补色。一种当做主色，另一种用于强调

三色

  • 间隔三色组 取彼此等距的三种颜色，就是此三种颜色在色轮相距的距离是一样的
  • 分类互补三色组 从色轮选择一种颜色，再在色轮上找到它对面的互补色，取互补色两侧的颜色

亮色/暗色

  • 上面提到的纯色就是色调
  • 向色调增加黑色就构成一个暗色
  • 向色调增加白色就构成一个亮色

最佳实践：

  • 单色组合 由一种色调及其相应的多种亮色和暗色组成
  • 暗色和亮色的组合 在上面三色方案中，不使用色调，而是使用这些颜色的暗色和亮色

暖色/冷色

暖色（包含红色或黄色），冷色（包含蓝色）。冷色趋于做背景色，而暖色是趋进型的，容易引起注意。如何组合暖色和冷色，一定要少用写暖色。

CMYK/RGB

在计算机中，颜色由R(ed)G(reen)B(lue)组成。故显示在屏幕上的内容应使用RGB

CMYK代表Cyan（一种蓝色）、Magenta（一种红/粉红色）、Yellow（黄色）和一个Key色（通常是黑色），需要印刷的项目应该使用CMYK

字体

字体分为6类：

  • 旧式体(OldStyle)
  • 现代体(Modern)
  • 粗衬线体(Slab serif)
  • 无衬线体(Sans serif)
  • 手写体(Script)
  • 花体(Decorative)

设计考虑的方面：

  • 大小(Size)
  • 粗细(Weight)
  • 结构(Structure)
  • 形状(Form)
  • 方向(Direction)
  • 颜色(Color)

字体的基本规则:

  • 标点后一个空格
  • 永远不要使用下划线
  • 字距调整
