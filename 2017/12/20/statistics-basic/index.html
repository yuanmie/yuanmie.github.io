统计学之基本知识

发表于 2017-12-20 | 分类于 数学

这里是学习统计学的一些笔记.

总体:要研究的一组对象,人或事物

采样:从总体采集有代表性的样本,有随机采样与非随机采样

样本:总体的一个子集

均值:如果有一个n个值的样本\(x_{i}\)​.那么它的均值\(\mu\)就等于样本的总和除于样本的数量

\[ \mu = \frac{1}{n}\sum_{i}x_i \] 方差:样本的分散情况\(\sigma^2\) \[ \sigma^2 = \frac{1}{n}\sum_i(x_i - \mu)^2 \] 标准差:方差的平方根\(\sigma\)

分布**:样本中各个值出现的频繁程度

频数:样本中一个值出现的次数

概率:频数除于样本大小

概率质量函数(Probability Mass Function):将值转换为概率的函数\(PMF\)

累计分布函数CDF(Cumulative Distribution Function):PMF函数的累计,公式为: \[ CDF_X(x) = PMF(X < x) \] 归一化(Normalization):将数据进行转换,使其均值为0,方差为1

众数:分布中出现的最多的数

异常值:远离总数的值

条件概率:依赖于某个条件的概率.

指数分布:

帕累托分布:

正态分布:

概率:介于0到1之间的值,对应于定性的描述某一件事发生的可能性的大小\(P\).这应该是一个更通用的定义.

事件:被赋予概率的事情称为事件\(E\).

试验:检测事件发生情况的过程.

概率计算:假设有事件\(A\)和事件\(B\)

  • 若事件\(A\)和\(B\)是相互独立的 \[ P(AB) = P(A)P(B) \]

  • 若事件\(A\)和\(B\)不独立 \[ P(AB) = P(A)P(B|A) \]

  • 若事件\(A\)和\(B\)是互斥的 \[ P(A|B) = P(B|A) = 0 \]

  • 一般情况 \[ P(A或B) = P(A) + P(B) - P(AB) \]

伯努利试验:在同样的条件下,重复的,相互独立的进行的一种随机试验.其结果只有2种可能:发生(概率为\(p\))或不发生(概率为\(1-p\)).

二项分布:即重复\(n\)次的伯努利分布.

计算发生\(k\)次指定事件的概率: \[ PMF(k) = \binom{n}{k}p^k(1-p)^{n-k} \]

\[ \binom{n}{k} = \frac{n!}{k!(n-k)!} \]

蒙特卡罗模拟:模拟随机系统计算概率的过程.

贝叶斯定理:两个事件的条件概率之间的关系.条件概率写成\(P(A|B)\),表示的是在事件\(B\)已发生的情况下事件\(A\)发生的概率.\(P(A|B)\)称为后验概率,\(P(A)\)称为先验概率.\(P(B|A)\)称为似然值.\(P(B)\)称为归一化常量. \[ P(A|B) = \frac{P(B|A)P(A)}{P(B)} \] 概率密度函数PDF(Probability density function):累积分布函数的导数.表示的是一种概率密度,而不是概率.概率密度衡量的是\(x\)轴上每个单位的概率.为了得到随机变量\(X\)落在某个区间的概率,我们可以计算概率密度函数在这段区间的微分. \[ P(a \le x \le b) = \int^{a}_{b}PDF_X(X)dx \]

卷积:计算两个随机变量的和的分布.不同于卷积网络的卷积.

正态分布的性质:

若一个随机变量\(X\)服从参数\(\mu\)和\(\sigma\)的正态分布,记做 \[ X \sim \mathcal{N}(\mu, \sigma^2) \] 若\(X' = aX + b\),则 \[ X' \sim \mathcal{N}(a\mu+b, a^2\sigma^2) \] 若\(Z=X+Y\),且$X (_X, {_X}^2) \(和\) Y(_Y, {_Y}^2)$,则 \[ Z \sim \mathcal{N}(\mu_X+\mu_Y, {\sigma_X}^2+{\sigma_Y}^2) \] 中心极限原理:如果将大量服从某种分布的值加起来,所得到的和会收敛到正态分布.

假设随机变量X的均值和标准差为\(\mu\)和\(\sigma\),那么\(n\)个随机变量\(X\)的和渐进的服从\(\mathcal{N}(n\mu, n\sigma^2)\)

均方误差MSE(Mean Squared Error): \[ MSE= \frac{1}{m}\sum(\hat x - \mu)^2 \] 极大似然估计值:使得似然函数最大化的估计

残差:预测值与实际值的差——————-\(\epsilon = \hat y - y\)

最小二乘拟合:最小化残差的平方和————-\(\min\sum{\epsilon_i^2}\).
