<如何阅读一本书> 读书笔记

发表于 2017-11-28 | 分类于 notes

如书名所示,这是一本教人如何阅读的书籍.一个问题是:阅读书的技巧需要被教授吗?从小学开始,我们就一直做着阅读理解.

窗前明月光,疑是地上霜.举头望明月,低头思故乡.

这个传达了作者怎样的心情.或者阅读一些近现代文章,找出文章的主旨.

关于如何阅读一篇文章,诗词或古文,我觉得中国的学生都比较擅长.那么,关于阅读一本书呢,在我的教育中确实很少提及这方面(我接受的教育是这样,不知道更高级的教育是怎么样:P).关于一本书,它应该怎么被阅读.序言是干什么的?目录又有什么用?从头一页一页读到尾吗?读的速度如何决定?

让我们打开这本书,看看是否有真材实料.

阅读的层次

我们因何阅读?

为资讯而读,作者说的每一句,我们都能理解.这本书只是把我们之前不了解的东西传达出来,并没有增进我们的理解.比如阅读报纸

为求得理解而读,作者写的东西,超出我们目前的理解.我们凭借着自己的智慧,把玩着作者的文字,靠着努力从不理解到渐渐理解甚至完全理解.

阅读就是学习

主动式阅读 阅读是一个活动,越主动,效果就越好.主动是在阅读的过程中,我们运用自己的技巧,心智,思考,想象与记忆等来理解作者想传达的东西.求得理解而读 需要更主动,为资讯而读 所需的主动较少,你和作者有同等的理解水平,只需要接收资讯,不需要努力就能理解.

阅读的过程就是学习的过程 在阅读的过程中,我们能增长知识,提高我们的理解水平.当然,有时候我们只是单纯的为消遣而阅读,这也是一件非常幸福的事.

阅读的四个层次

基础阅读

检视阅读

分析阅读

主题阅读

这四个阶段是一层层递进,高层次包含低层次的.让我们来看看这四个层次分别在做什么.

基础阅读

简略的讲,基础阅读就是你认得书上的字了,这部分的阅读在我们小学阶段就搞定了.想一想,假如我们需要阅读一本梵文书,但是我们不懂,该如何阅读下去.(PS:有图画或许可以看图说话:P)

因此,当我们阅读一本书时,需要先搞定基础阅读这个阶段.

检视阅读

有系统的略读 先看看书的书名,副标题.看看书的序,翻翻目录,通过目录了解整本书的结构.然后东翻翻西翻翻,随便看几段或者连续看几页.这样我们对这本书已经了解很多了.

粗浅的阅读 当我们读一本高难度的书时,遇到难懂的地方应该怎么做?是停滞不前,然后想到这本书可以不适合我.还是先跳过它,继续阅读下去.

答案是后者.跳过不懂的地方,读完整本书.即使只有50%的理解也好过卡住导致一无所知.而且能在你重读时帮助你理解那些难懂的地方.

阅读的速度 不是每本书都是恒定的速度去阅读.对于那些就需要略读的书籍,尝试较快的速度.对于那些精读的书籍,那就放慢速度,尝试更深刻的理解它.这个的关键是:对于不同的书籍,我们需要知道该以怎么的速度去阅读.

一个阅读者要提出的四个基本问题 就像老师一直教的,要带着问题去阅读.在阅读一本书时,我们需要带着这些问题,并且在读完后能得到答案.

  1. 整本书在讲什么?
  2. 在局部上,作者说了什么,如何表达的?
  3. 这本书说的东西有道理吗?是全部有道理,还是部分有道理?
  4. 这本书和我们有什么关系

分析阅读

分析阅读是本书着墨最多的部分,也是常用的阅读技巧.我们一般阅读的时候,都是在这个阶段.(有可能在检视阅读发现这本书不值一读,那就没有然后了:P)

分析阅读也分几个大阶段,每个阶段又分为几条规则.这些阶段应该是一次阅读一起完成的,而不是第一遍阅读完成第一阶段,第二遍阅读完成第二阶段之类的.

所幸<如何阅读一本书>总结的非常好,让我们一起来看一看:P.

第一阶段

  1. 依照书本的种类与主题分类. 在上一个阶段,我们基本可以知道要阅读的书的分类,是诗集还是科学类.
  2. 用最简短的句子说出整本书在谈什么. 用自己的话语,而不是书上写的辞藻来说出主旨,理解的深刻才能简短.
  3. 按照顺序与关系,列出全书的重要部分.将全书的纲要拟出来之后,再将各个部分的纲要也一一列出.
  4. 找出作者在问的问题, 或作者想解决的问题. 找准了问题,就能更好的理解答案(书的内容就是作者给出的答案)

第二阶段

  1. 诠释作者使用的关键词,与作者达成共识. 文字是有歧义的,不像数学定义只有唯一的意思.找出关键词,确保和作者理解的是同样的意思.
  2. 从最重要的句子中找出作者的重要主旨. 书的开头和结尾是找最重要的句子的好地方.
  3. 找出作者的论述, 重新架构这些论述的前因后果,以明白作者的主张
  4. 确定作者已经解决了那些问题, 还是哪些是未解决的.在为解决的问题中,确定哪些是作者认为自己无法解决的问题

第三阶段

此阶段当你阅读完一本书,评论一本书的规则

评论的礼节:

  1. 除非你已经读懂了,否则不要轻易批评
  2. 不要争强好胜,非辩到底
  3. 在说出评论之前,要能证明自己区别的出真正的知识与个人观点的不同

批评观点的标准

  1. 证明作者的知识不足
  2. 证明作者的知识错误
  3. 证明作者不合逻辑
  4. 证明作者的分析与理由是不完整的.

主题阅读

主题阅读,就是阅读的终极目标了,大佬们才能触及到的神之领域.想象大佬们针对一个主题,纵览群书,取其精华,融自我之智慧,孕超我的思想.OK,下面看看主题阅读如何实践.

一 准备阶段

  1. 针对要研究的主题,粗糙的拟定一份书单.
  2. 利用检视阅读,确定哪些书与主题相关.在这过程中,对研究的主题建立起清晰的概念.

二 阅读

  1. 阅读准备阶段的书,找出最相关的章节(注意:主题阅读不会每本书都从头看到尾)
  2. 根据主题创建出一套中立词汇.
  3. 列出一个中立的主旨,列出一连串的问题.看看这些作者的解答如何.
  4. 界定主要及次要的议题,然后将作者针对各个问题的不同意见整理陈列在各个议题旁边.
  5. 分析这些讨论.形成自己的思想.

阅读与心智

一言以蔽之,阅读带来心智的成长.

读一本好书,会让你的努力有所回报.下面摘自原文:

第一:当你成功的阅读了一本难读的好书后,你的阅读技巧必然精进了.

第二:一本好书能教你了解这个世界以及你自己.你不只更懂得如何读得更好,还更懂得生命.你变得更有智慧,而不只是更有知识.
