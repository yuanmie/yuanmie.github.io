写个大家看的设计书·浅读

发表于 2018-05-20 | 分类于 notes

设计的四个基本法则：

亲密性

彼此相关的项应当靠近，归为一类。如果多个项相互之间存在很近的亲密性，它们会称为一个视觉单元，而不是多个孤立的元素。这有助于组织信息，减少混乱，为读者提供清晰的结构。

对齐

任何东西都不能在页面上随意安放。每个元素都应当与页面上的另一个元素有某周视觉联系。这样能建立一种清晰、精巧而且清爽的外观。

重复

让设计中的视觉要素在整个作品中重复出现。可以重复颜色、形状、材质、空间关系、线宽、字体、大小和图片，等等。这样一来，既能增加条理性，还可以加强统一性。

对比

要避免页面上的元素太过相似。如果元素（字体、颜色、大小、线宽、形状、空间等）不相同，那么就干脆让他们截然不同。要让页面引人注目，对比通常是最重要的一个因素，正是它能使读者首先看这个页面。

JUnit5

发表于 2018-04-05 | 分类于 Java

前言

JUnit 是被广泛使用的测试框架，主要用于单元测试。

在计算机编程中，单元测试（英语：Unit Testing）又称为模块测试, 是针对程序模块（软件设计的最小单位）来进行正确性检验的测试工作。程序单元是应用的最小可测试部件。在过程化编程中，一个单元就是单个程序、函数、过程等；对于面向对象编程，最小单元就是方法，包括基类（超类）、抽象类、或者派生类（子类）中的方法。

https://www.wikiwand.com/zh-hans/%E5%8D%95%E5%85%83%E6%B5%8B%E8%AF%95

使用 JUnit 的好处主要有那么几点：

  • 可以使测试代码与产品代码分开。
  • 针对某一个类的测试代码通过较少的改动便可以应用于另一个类的测试。
  • JUnit 是公开源代码的，可以进行二次开发。
  • 可以方便地对 JUnit 进行扩展。

专注

发表于 2018-04-01 | 分类于 notes

前言

我有手机成瘾的倾向，不管有事没事，我都会每隔几分钟（甚至是1，2分钟）拿起手机，打开TIM，微信，把所有的菜单都点一遍。是的，没消息我也会全点一遍，仿佛这样就会冒出几条消息。之前用的夸克浏览器，我也会把它推荐的应用点开，看看有没有新的文章。更无聊的是，有时候会把设置选项点一点，新大陆等着我去发现。

正如之前《自制力》提过的，多巴胺给了我虚假的承诺–去玩手机吧，你会得到快乐。不，实际上并没有。玩后剩下的是空虚，罪恶感。我想把这些时间收集起来，来做自己感兴趣的事。这就是我想推荐这本书的理由。

可能，你在想，这本书有这么厉害吗，看了就能改掉上述的坏习惯？答案是NO。这本书只是一个引子，它给予我们一些建议，然而真正掌控一切的是我们自己，需要看书，思考，根据自己的实际情况实践。

实际上，这是我第三次看这本书。这是一本小而精的电子书，读一次大概需要2小时，书本的排版（kindle)也非常舒服。每一次我看完后，都会被书中的生活方式所折服。专注，多么美好,进入心流状态。我勤而行之，遗憾的是，随着记忆的遗忘，慢慢的那些坏习惯又回来了。虽然过程是如此的反复，但是并非没有效果。我现在吃饭也不玩手机了，习惯性打开微信的频率也降低了。虽然步伐很小，但我还是向前迈了几步。

当你感觉自己正在被外界纷繁之物所干扰，找块空白时间，重新读读这本书。好书是值得反复阅读的。

七堂极简物理课

发表于 2017-12-27 | 分类于 notes

前言

这是非常好的科普书,极大的颠覆我的认知.

从人类诞生起,就对世界充满了好奇.我们总是抬起头,仰望星空.思考着:天地是什么形状,人类是来自哪里,时空是什么东西.之前看过一句古文:

​ 四方上下曰宇, 往来古今曰宙

在古时,人们已经通过想象去猜测世界,类似的还有天圆地方,五行论.随着科技的发展,我们首次认识到地球是圆的,提出了地心说,后来哥白尼提出了日心说,再后来我们知道太阳系也是宇宙的一隅中的一个小角落.牛顿提出了万有引力定理,我们知道了为什么我们不会飞起来(好想飞),开创了现代物理.马克斯·普朗克提出了量子论,正式进入量子物理时代.

人类在探索世界的路途上一步一步前进.我们还没有彻底的揭开那层神秘的面纱,但是这路途上的风景是如此波澜壮阔,绚丽缤纷.

在广义相对论中,我们知道空间即引力场,也是一种物质.在量子物理中,我们知道了原子的存在,能量并不连续,世界基于事件.亦会诧异整个宇宙只有少数几种粒子构成–电子,夸克,光子,胶子,中微子以及暗物质.空间并不无限可分,空间也是由空间粒子构成.热量可能会成为时间的关键,概率被引进了物理世界.认识到在这个浩瀚的宇宙中,人类只是自然亿万万可能中的一种.我们本身就是自然一部分,并不是旁边者.我就是我.
