思考 快与慢

发表于 2018-08-27 | 分类于 notes

这个可能是我看的最艰难的一本书。当然，看到了很多有趣的东西，我觉得是非常值得。

书本包含的东西较多，在此只做简短的笔记。

Part 1 系统1， 系统2

我们给大脑分配2种虚拟角色，分别是系统1和系统2。系统1负责直觉之类，系统2负责更深入的思考。我们大部分时间都用系统1，比如红绿灯，吃饭等等。当系统1碰到难题时，才会求助系统2，而系统2是懒惰的。而系统1并不是时刻都正确，有时候我们会做出错误的决定。

联想的神奇力量

联想是系统1的一个运行过程，而且都是自主发生的，无需费力。我们看到，接触的事物会引发我们的联想，这份联想继而会影响我们的行为。比如你看到一些描述老年的词汇，让你想到了老年，你走路的步伐会无意识的变慢。

常见的动作会不知不觉的影响到我们的想法和错觉。比如当你觉得高兴时会微笑，而微笑也会使你高兴。

人生算法

发表于 2018-08-05 | 分类于 notes

所有有意义的改变都是从内到外的

本文作为《人生算法》一书的笔记，书中描述了15个有效的人生算法。这些算法确实很有道理，但最重要的是时刻实践和应用。

  1. 戴上正确的眼镜

    审视我们看待自己与他人的方式是否正确。不要被先入为主或固有的印象所禁锢，正确的，客观的用心去看待自己和他人。

  2. 掌控自己的心理状态与情绪

    不让外界因素决定我们如何思考与感受。正如范仲淹说过：不以物喜，不以己悲。要达到这种境界需要一种非常豁达的心态。

  3. 行事为人让他人能够信任

    展示自己的品格和能力，要有优秀的品格（如诚信）和目前所适应的能力，别人才会相信你。

    眼光要放长远，罗马不是一天建成的，信任的构建也需要时间，它是一点一点累计起来的。

    适应当前的情况，与不同的人建立信任的方法不同，找到合适的方法。

枪炮、病菌与钢铁

发表于 2018-07-29 | 分类于 notes

整本书都在回答一个问题：为什么是欧亚大陆征服了其他大陆，而不是相反的结果？

非洲大陆是人类的发源地。古人从非洲大陆出发，现在已遍布世界（除两极）。各个大陆的人在身体构造上、智力上并没有区别。几万年过去了，是什么导致了文明与落后？

最初人类是一个狩猎民族（现代社会,在一些极度隔离的地区还存在一些狩猎民族），以打猎为生，居无定所。人类最大的转折是粮食的出现。粮食促进了人口的快速增长，人口的快速增长促进了更多的粮食需求，这是人类的胜利，也是粮食的胜利。

粮食的发展，使得人类过上了定居生活。人口的大量生长与聚集，导致行政制度的产生，目前最大的行政制度就是国家。正是有了多余的粮食和这些制度，才出现了科学家，艺术家等非农民的职业，才能出现科学技术与艺术的发展，才有之后的火药，文字等等工具。

读毛姆文集

发表于 2018-07-19 | 分类于 notes

满地都是六便士，他却抬头看见了月亮

初识毛姆，是通过《阅读是一座随身携带的避难所》，接着读了《月亮与六便士》。毛姆的笔触似乎有种魔力，吸引读者一直读下去，它的文字朴实却不失华丽，舒服却不失深刻。毛姆的代表作有《人性的枷锁》、《月亮与六便士》。

我是偶然间看到《阅读是一座随身携带的避难所》的。在过了一段时间之后，我才理解书名。这个时代的我们太焦虑了，高不可攀的房价，源源不断的欲望，让我们在现实中过的很痛苦，憎恨它并逃避它。有阅读的时光，让焦躁的心灵沉静下来，让不安的灵魂脱离尘世，进入到书籍的世界，达到忘我的境界。毛姆精心挑选了一些名著并介绍了一些作者的故事，读来十分有趣。毛姆认为—–小说家的使命是写出好故事，他们不需要是哲学家，要对人生做出深刻的见解。

人生的智慧·浅读

发表于 2018-05-31 | 分类于 notes

我认为决定凡人命运的根本差别在于三项内容：

（1）人的自身，包括人的健康、力量、外貌、气质、道德品格、精神智力及潜在发展

（2）人所拥有的身外之物，亦即财产及其他占有物

（3）人向其他人所显示的样子，即人们对他的看法

对于造成人们的幸福抑或不幸福，第一项会产生更为根本和彻底的影响，因为后两项出自人为的划分，而第一项是大自然确定的。

先说结论，刻意练习是一种有效的学习方式。刻意练习是什么？简短的说，通过刻意的进行练习，来达到提高技能水平的活动。刻意练习将人们推向学习区，在挑战面前，人们专注的思考，行动去解决它，然后得到反馈，自己的思路或方法是否正确，是否有改进之处，接着不断的纠正。如此循环，一步步攻克挑战，将技能水平提高。
